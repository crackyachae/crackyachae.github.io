Vim 사이트의 documentation 페이지에서 도움이 될 만한 여러 사이트들을 소개하고 있다. Vim을 학습하면서 추가적으로 알게된 사이트들을 함께 소개한다.

## Vim Help

Vim의 online documentation은 [Vim help](https://vimhelp.org/)에서 제공한다. Vim의 [FAQ](https://vimhelp.org/vim_faq.txt.html)가 Vim help에 들어있다.

홈페이지에서는 소개하고 있지 않지만 다음 사이트가 vim에 처음 접근하기에 좋은 것 같아 별도로 소개한다.

+ [Vim Quick Start 가이드](https://www.joinc.co.kr/w/Site/Vim/Documents/quick_start)
+ [Vim, 두 가지 관점](https://johngrib.github.io/wiki/two-views-of-vim/#초심자는-vim을-어떻게-공부하는-것이-좋을까) by 기계인간 (John Grib) 
+ [초보자를 위한 Vim 입문서](https://www.joinc.co.kr/w/Site/Vim/Documents/UsedVim)
  
## Vim Adventure

[Vim Adventure](https://vim-adventures.com/)에서 게임으로 Vim을 학습할 수 있다. 총 12개의 스테이지로 되어있고 3단계 이후로는 6개월 단위의 라이센스 (25$)를 구매해서 이용해야한다.

Vim Adventure 외에도 vim을 직접 사용해보면서 학습할 수 있는 사이트들이 있다.

+ [Openvim](https://www.openvim.com/tutorial.html): Vim의 기본 기능들을 interactive하게 배울 수 있는 사이트. Practice 페이지에서 배운 내용을 자유롭게 연습해볼 수 있다.
+ [Vim Genius](http://vimgenius.com/): Flashcard 형식으로 Vim의 기본 기능들을 학습할 수 있는 사이트. 총 7단계로 되어있고 각 단계마다 10개 내외의 명령을 학습한다.

## Vim Tips

[Vim Tips](https://vim.fandom.com/wiki/Vim_Tips_Wiki)에서 Vim을 더욱 편하게 사용하거나 문제를 빠르게 해결 할 수 있는 다양한 팁을 찾을 수 있다.

## Other Documentations

[Vim Docs](http://vimdoc.sourceforge.net/)와 [Vim Web Pages](https://www.vim.org/web.php)에서는 다양한 Vim documentation 링크를 모아서 제공하고 있다. 개인이 제작한 튜토리얼과 Vim help 번역본 링크를 제공하고 있지만, 특정 시점 이후로 업데이트되어있지 않다. 

[한국어 번역](https://vim-ko.github.io/doc/help.html)도 있지만 이 역시 2012년도 이후로 업데이트되어있지 않다.

Vim 7.3 버전을 기준으로 제작한 [Vim user manual](http://www.eandem.co.uk/mrw/vim/usr_doc/index.html)과 [Vim book](http://www.truth.sk/vim/vimbook-OPL.pdf)의 PDF파일을 받을 수 있다.

+ [Vim Cheatsheet](https://vim.rtorr.com/lang/ko/)
