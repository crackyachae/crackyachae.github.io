```python
while True:
    a, b = map(int, input().split())
    if a == 0:
        break
    print(["No", "Yes"][a > b])
```

### 풀이 과정

* 종료가 아닌 입력은 양의 정수만 들어오기 때문에 마지막 입력 체크 조건을 `a == 0`만 했다.
    * `a == 0 and a == b`로 하면 조건에 부합한다.
    * `not (a + b)`가 제일 깔끔할 것 같다.
