```python
a, b = map(int, input().split())
print(a + b, a - b, a * b, a // b, a % b, end="\n")
```

### 아이디어 & 풀이

입력받은 `a`, `b`를 연산한 값을 출력한다.

* `end` 값을 `\n`으로 주면 `print()`문 한 줄로 연산 결과를 각각의 줄로 출력할 수 있다.
