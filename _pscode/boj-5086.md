```python
while True:
    a, b = map(int, input().split())

    if a == 0 and b == 0:
        break
    if a % b == 0:
        print("multiple")
    elif b % a == 0:
        print("factor")
    else:
        print("neither")
```

### 풀이 설명

* 나머지가 a를 b로 나눈 나머지가 0일 때, a는 b의 multiple, b는 a의 factor이다.
