## 2021.03.25

| 메모리    | 시간  | 코드 길이 |
| --------- | ----- | --------- |
| 28776 KB  | 68 ms | 68 B      |

```python
K, N, M = map(int, input().split())
print([0, K * N - M][K * N > M])
```

### 풀이 과정

* `K * N > M` 조건이 참이면 1 거짓이면 0을 반환하는 것을 리스트 인덱스로 이용한다.

## 참고 답안

```python
# 풀이 1
a, b, c = map(int, input().split())
d = a * b - c
d = int(d)
if d >= 0:
    print(d)
else:
    print("0")

# 풀이 2
a, b, c = map(int, input().split())
print(max(0, a * b - c))
```

### 참고 사항

* 경우가 간단해서 if문을 이용해서 정석적으로 풀어도 괜찮은 것 같다.
* 필요한 돈 보다 가진 돈이 많을 경우 결괏값이 음수가 나오기 때문에 `max`로 값을 결정하도록 해도 된다.
