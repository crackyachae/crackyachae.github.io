## 2021.03.04

| 메모리    | 시간  | 코드 길이 |
| --------- | ----- | --------- |
| 28776 KB  | 68 ms | 94 B      |

```python
a, b, c = map(int, input().split())
print((a + b) % c, (a % c + b % c) % c, (a * b) % c, (a % c * b % c) % c, sep="\n")
```

### 아이디어 & 풀이

주어진 대로 연산한 뒤 출력한다.

* `sep` 값을 `\n`으로 주면 `print()`문 한 줄로 연산 결과를 각각의 줄로 출력할 수 있다.
