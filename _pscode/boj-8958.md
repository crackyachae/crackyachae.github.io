```python
n = int(input())

for _ in range(n):
    s = input()
    # 총 스코어
    sc = 0
    # 연속된 횟수
    now = 0
    for i in s:
        # "O"일 때마다
        if i == "O":
            # now를 증가시킨 뒤
            now += 1
            # sc에 더한다.
            sc += now
        # "O"가 끊어지면
        else:
            # now를 0으로 초기화한다.
            now = 0
    print(sc)
```

### 풀이 과정

* 연속된 횟수를 세는 변수와 총 스코어를 저장하는 변수를 따로 생성해
* O가 나오면 연속된 횟수를 1 증가시킨 뒤 이를 스코어에 더하는 과정을 반복한다.
* X가 나오면 연속된 횟수를 초기화한다.

## 참고 답안

```python
import sys

n = int(input())
for i in range(n):
    ans = sys.stdin.readline().rstrip()
    res = 0
    # X로 split한 각 원소에 대해서
    for j in ans.split("X"):
        # 연속될 때마다 1씩 증가하면서 더하기 때문에
        # O가 n개 있다면 점수는 1 + ... + n = n * (n + 1) / 2
        k = j.count("O")
        res += k * (k + 1) / 2

    print(int(res))
```
