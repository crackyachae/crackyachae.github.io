## 2021.04.14

| 메모리    | 시간  | 코드 길이 |
| --------- | ----- | --------- |
| 28776 KB  | 72 ms | 62 B      |

```python
N = int(input())

for i in range(N, 0, -1):
    print("*" * i)
```

### 아이디어 & 풀이

`i` 순서를 5부터 역순으로 순회하기 위해 `range()`의 시작을 N, 끝을 0, 단계를 -1로 입력한다.

## 참고 답안

```python
n = int(input())

for i in range(n):
    # 전체에서 현재 i 값을 뺀 개수만큼 별을 출력한다.
    print("*" * (n - i))
```
