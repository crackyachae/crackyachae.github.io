## 2021.04.13

| 메모리    | 시간   | 코드 길이 |
| --------- | ------ | --------- |
| 28776 KB  | 156 ms | 107 B     |

```python
T = int(input())

for _ in range(T):
    # 입력받은 값으로 리스트를 만들어
    A = list(map(int, input().split()))
    # 정렬한 뒤
    A.sort()
    # 뒤에서 세 번째 원소를 출력한다.
    print(A[-3])
```

### 아이디어 & 풀이

세 번째로 큰 수를 찾기 위해서는 리스트를 정렬한 뒤, 뒤에서 세 번째 수를 찾으면 된다.

### 피드백

* 처음 배열을 생성할 때 `sorted`를 써서 만드는 게 나을 것 같다.
