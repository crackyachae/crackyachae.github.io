## 2021.04.14

| 메모리    | 시간  | 코드 길이 |
| --------- | ----- | --------- |
| 28776 KB  | 64 ms | 87 B      |

```python
N = int(input())

for i in range(1, N + 1):
    print(" " * (N - i) + "*" * (2 * i - 1))
```

### 디버그

* 대칭을 맞춘답시고 `*` 이후에도 공백을 출력하도록 했는데 그럴 필요가 없었고 정답처리도 되지 않았다 (결과: 출력 형식이 잘못되었습니다).
