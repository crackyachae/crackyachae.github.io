| --------- | -----  | --------- |
| 28776 KB  | 124 ms | 264 B     |

```python
T = int(input())

for _ in range(T):
    # 0으로 초기화해둔 `ys`, `ks`에
    ys = ks = 0

    # 9번에 걸쳐 입력받은 값을 더한 뒤
    for _ in range(9):
        y, k = map(int, input().split())
        ys += y
        ks += k

    # 이 둘을 비교해 결과를 출력한다.
    if ys > ks:
        print("Yonsei")
    elif ys < ks:
        print("Korea")
    else:
        print("Draw")
```
