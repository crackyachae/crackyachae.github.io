```python
b = input()
h = 10

for i in range(1, len(b)):
    if b[i] == b[i - 1]:
        h += 5
    else:
        h += 10

print(h)
```

### 풀이 과정

* 이전의 값과 현재 값이 같으면 5를, 다르면 10을 더한다.
* 현재값과 전의 값을 조회하기 때문에 인덱스는 1부터 시작하고 초깃값을 10으로 잡는다.
