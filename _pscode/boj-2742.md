## 2021.04.14

| 메모리    | 시간  | 코드 길이 |
| --------- | ----- | --------- |
| 36732 KB  | 84 ms | 60 B      |

```python
n = int(input())
print("\n".join(map(str, range(n, 0, -1))))
```

### 풀이 설명

* `range()`가 반환하는 숫자를 역순으로 만들기 위해 시작을 주어진 값, 끝을 0, 단계를 -1로 지정한다.
* `range()`가 반환한 이터레이터를 `join()`을 이용해 `\n`으로 결합해서 한 번에 출력한다.
* `join()`을 쓰기 위해서는 순회하는 값들이 문자열이어야 하므로 `str`으로 매핑해준다.
