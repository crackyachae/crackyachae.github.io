```python
charge = 1000 - int(input())
count = 0
for coin in [500, 100, 50, 10, 5, 1]:
    num = charge // coin
    count += num
    charge -= coin * num

print(count) 
```

### 아이디어 & 풀이

가장 적은 수의 거스름돈을 내기 위해서는 액수가 큰 잔돈으로 낼 수 있는 최대의 금액을 치러야 한다.

* 500엔부터 내림차순으로 잔돈을 순회하면서
* 나눈 몫을 개수에 더하고
* 해당 금액을 내야 할 금액에서 빼주면 된다.
