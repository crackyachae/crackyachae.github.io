```python
N = int(input())
# 입력받은 수를 리스트로 만들어
a = list(map(int, input().split()))
# 정렬한 뒤 
a.sort()
# 첫 번째 원소와 마지막 원소를 출력한다.
print(a[0], a[N - 1])
```

### 풀이 과정

* 리스트를 정렬하면 첫 번째 값이 최솟값, 마지막 원소가 최댓값을 갖는다.
* 입력받은 수를 리스트로 만들어 정렬한 뒤, 첫번째와 마지막 원소를 출력한다.
