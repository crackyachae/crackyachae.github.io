```python
import sys
from bisect import insort_left
input = sys.stdin.readline

L = []
for _ in range(int(input())):
    insort_left(L, int(input()))

print(*L, sep="\n")
```

## 참고 답안

```python
L = []
for _ in range(int(input())):
    L.append(int(input()))
L.sort()

print(*L, sep="\n")
```

### 아이디어 & 풀이

`sort()`를 사용한다. 속도는 크게 차이 나지 않는 것 같다.
